# Third-party data and model-artifact boundaries

This repository does not redistribute LaDe or the Amazon Last Mile Routing Research Challenge data.

- **LaDe**: obtain the files from the official project, accept its research-use terms, and normalize
  delivery records according to `docs/lade_adapter.md`.
- **Amazon Last Mile Routing Research Challenge**: the public dataset is provided under
  CC BY-NC 4.0. Preserve attribution and noncommercial restrictions. The files generated by
  `make amazon-smoke` are synthetic and only mimic the official JSON shape.

Serialized Joblib and PyTorch files are executable/trusted-code artifacts. Load only artifacts
produced by this repository or another trusted source. SHA-256 manifest verification detects
accidental corruption but is not a substitute for cryptographic signing or supply-chain controls.
